# =============================================================================
# SilvaEngine Gateway — container dependency set
# =============================================================================
# The gateway is a single FastAPI/uvicorn app that serves MCP (JSON-RPC), REST,
# GraphQL, and SSE on one port. Private ideabosque repos are fetched over SSH
# (the image copies a deploy key and trusts github.com — see Dockerfile).
#
# The ENGINE MODULES (knowledge_graph_engine, rfq_engine, mcp_daemon_engine)
# are NOT installed here. They are bind-mounted from host source into /app/src
# and put on PYTHONPATH (see docker-compose.yml + SILVAENGINE_SRC). This file
# installs only their THIRD-PARTY dependencies, the shared SilvaEngine libs, and
# the gateway. Keep the engine-deps section in sync with each engine's
# pyproject `dependencies` when they change.
# =============================================================================
